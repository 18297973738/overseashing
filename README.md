##项目中所遇到的问题级解决方案
-1.访问地址时出现跨域问题
解决方案:config/index.js这个文件里面添加对应的http代理
-2.搜索商品时，关键字使用
解决方案:通过input中双向数据绑定v-model解决
-3.获取数据详情时,通过地址栏绑定动态数据,刷新时出现问题
解决方案:通过地址栏动态地址判断数据应该取哪一条
